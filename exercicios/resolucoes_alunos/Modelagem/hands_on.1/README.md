# Primeiro exercicio Hands On

## Criação de exemplos de modelagem conceitual, relacional e fisica no formato PDF.

### Imagine uma loja de **reparos** de instrumentos, onde  **clientes** levam seus **instrumentos** para serem reparados. Como você faria a modelagem para solução deste problema?

Seja criativo, mas **atenção**, não fuja do contexto. Uma tabela para uma determinada entidade tem que ser coesa.

### Outputs esperados deste exercício:

- Geração de pelo menos 3 arquivos `.pdf` com a modelagem:
    - Conceitual
    - Relacional
    - Fisica
